ASP.NET MVC Basic Authentication Sample
-----------------------------------
This sample demonstrates a custom authentication filter with MVC. It uses the
HTTP Basic Access Authentication scheme defined by RFC 2617 combined with the
ASP.NET Identity system for storing user names, passwords, roles and claims.

The sample automatically populates the Identity database with a sample user.
To login, run the project and provide the following user name and password
when prompted by your web browser:

SampleUser
secret
