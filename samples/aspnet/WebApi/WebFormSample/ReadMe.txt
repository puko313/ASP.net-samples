WebFormSample
-------------

This sample shows an ApiController added to a WebForm project using the menu 
"Add New Item" and then select "Web API Controller Class". In addition to the 
controller we also add a default route to global.asax.cs file. 

For a detailed description of this sample, please see
http://blogs.msdn.com/b/henrikn/archive/2012/02/23/using-asp-net-web-api-with-asp-net-web-forms.aspx
